

　　　　TSMemory ver0.4.0.1
　 　　（H.264/AVC、H.265/HEVC　対応版)


TSMemoryは、TVTestとAviUtlを連携させて映像メモリ機能を実現するためのプラグインセットです。
静止画キャプチャ用途での使用を想定しmpeg2-ts,H264（スカパープレミアム）,H265（4Kのみ）が対応しています。

現在、テスト用の人柱版ですが
下記にて動作確認しています。

　TVTest　0.10.0.0
　AviUtl　1.00
　L-SMASH　r1281 

必要ソフト

　TVTest
　AviUtl
　TSMemory
　L-SMASH（事前に導入をお願いします。）

導入と設定方法

1. TVTestのPluginsフォルダに TSMemory.tvtp と TSMemory.ini を入れます。
2. TSMemory.ini ファイルを開いて、メモリのサイズと AviUtl のパスを設定します。
3. AviUtlのフォルダに CaptureUtil.auf　と　TVTest_Image.dll（TVTest付属ファイル）を入れます。
   画像の保存する際に使います。
4. AviUtlを起動して下記３つを対応をする。
　 [入力プラグイン優先度の設定]からL-SMASH Works File Readerを一番上に設定します。
　 [入力プラグインの設定] → [L-SMASH Works File Readerの設定]から[threads]1、[seek mode]unsafe、[Cashe directory path]%TEMP%\　に設定します。
   [システムの設定] から[最大画像サイズ]3840x2160、[Large Address Aware]有効　に設定します。 
5. TVTest を起動し、メニューの [設定] で設定ダイアログを表示させ、
   左のリストから [キー割り当て] を選択して、TSMemoryの実行にキーを割り当てます。
6. メニューの [プラグイン] から、TSMemoryをチェック状態にします。

使い方

1. キャプチャしたいシーンになったら、導入の 6 で割り当てたキャプチャ実行キーを押します。
   AviUtlが起動して、映像メモリの内容が読み込まれるはずです。
2. AviUtlの表示メニューから「キャプチャ・ユーティリティ」を選択すると、
   キャプチャ・ユーティリティのウィンドウが表示されます。
   上段に任意の保存先+画像のファイル名を入力します。
　（例）デスクトップのTvtestというフォルダの場合　　C:\Users\**\Desktop\TVTest\image
   ファイル名の拡張子が省略されている場合は拡張子が付加されます。
   既に同名のファイルが存在する場合は、ベース名の末尾に連番が付加されます。
   下段左端でフォーマットをBMP、JPEG、PNGの中から選択、その右側にある圧縮レベルを指定します。
3. 「保存」ボタンを押すと画像がファイルに保存されます。
4. 以後、キャプチャ実行キーを押すと AviUtl で読み込まれます。


ヒント

・画像の保存時にファイル名をクリップボードにコピーしたい場合は、CaptureUtil.ini
  ファイルに CopyFileName=1 を追加してください。


注意

・8K放送は今のところ対応していません。
・このverからTVTestsrc一式が不要です。
・このプログラムによるいかなる損害も補償しません。
・音声には対応していません。

動作原理

映像メモリ機能を実行すると、TVTestが受信しているTSデータを
共有メモリから一時ファイル（%TEMP%\tsm????.ts）として書き出し、
AviUtlにWM_DROPFILESメッセージで渡します。

AviUtlはL-SMASH Works（lwinput.aui）を使ってTSファイルを読み込みます。
一時ファイルは次回実行時またはTVTest終了時に自動削除されます。
